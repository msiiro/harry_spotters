# The Vanishing Cabinet
This is a project for Grace and Andrea. It is meant as a place for them to list all of the Harry Potter fanfic that they have read or are reading. 

This is not something that just anyone can access. If you are reading this (and you are not Matt, Grace, or Andrea) then you have stepped dangerously into a place that you do not belong. PLEASE LEAVE NOW!!!

The name of the project is **The Vanishing Cabinet**
Some of the key features that were requested are:
- Integration with AO3 website: pulling descriptions, tags, characters, and other metadata
- Custom tags that they have defined for themselves
- Rating system for differnet criteria
- Their own reviews, both with and without spoilers
- Progress tracker they can use to compare how much they have read, likely with some sort of time component.
